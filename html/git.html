Основные команды Git

  • git init - инициализация нового репозитория Git в текущей директории
  • git clone [url] - клонирование удаленного репозитория на локальный компьютер
  • git status - показывает состояние рабочего каталога и индекса
  • git add [файл] - добавление файлов в индекс (staging area) для следующего коммита
  • git commit -m "сообщение" - создание коммита с изменениями из индекса
  • git push - отправка локальных коммитов в удаленный репозиторий
  • git pull - получение изменений из удаленного репозитория и их слияние с локальной версией
  • git branch - работа с ветками (просмотр, создание, удаление)
  • git checkout [ветка] - переключение между ветками
  • git merge [ветка] - слияние указанной ветки с текущей
  • git log - просмотр истории коммитов
